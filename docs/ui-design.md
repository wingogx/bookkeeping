## 🎬 动效设计

### 记账成功动画
1. 金币落入钱袋（0.3s）
2. 绿色涟漪扩散（0.5s）
3. 数字滚动更新（0.3s）
4. 轻微震动反馈

### 页面转场
- Push/Pop：标准iOS滑动（0.3s）
- Modal：底部弹出（0.25s）
- Tab切换：淡入淡出（0.2s）

### 加载动画
- 下拉刷新：标准弹性动画
- 数据加载：骨架屏渐现
- 语音识别：波形律动

这些UI设计充分考虑了用户体验，确保操作简单直观，视觉友好温暖。
